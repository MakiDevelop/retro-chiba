1989 年 9 月，Atari 在北美推出 Lynx，售价 $179.95 美元。**这是史上第一台彩色掌机**——与初代 Game Boy 同月发行、比 Sega Game Gear 早一年。原始设计来自 **Epyx**（《California Games》《Summer Games》系列开发商，当时的运动模拟游戏一线厂），原型机代号 *Handy*，1989 年 Epyx 财务危机后 Atari 接手量产。

技术上 Lynx 领先同期 Game Boy 一个世代。**MOS 6502 派生 CPU @ 4 MHz** + 自研协处理器 **Suzy / Mikey**（前者负责 blitter 图形运算、后者整合系统总线与音源），**4,096 同屏色彩色背光 LCD（160×102）**、64 KB RAM、4 通道立体声、**ComLynx 串行连线端口（最多 8 台同时连线）**。独特的硬件设计还包括**双向手柄设计**（通过屏幕翻转，左撇子可以镜像握持主控制）—— Game Boy / Game Gear / 后续所有掌机都没做这件事。

**最大命门是耗电**——**6 颗 AA 撑 4-5 小时**（vs Game Boy 4 颗 AA 撑 30 小时），这是彩色背光在 1989 年硬件成本下的物理限制（Game Gear 一年后也撞上同样问题）。Atari 1991 年推出 Lynx II（更小、立体声耳机输出、可关背光省电），但根本问题未解。**Atari 公司本身正在崩塌**——1989 年 Atari Corp 已被任天堂 NES（家用机 90% 市占）和 Sega Genesis 双重夹击，研发资源不足、第三方信心崩溃。

软件阵容稀薄是 Lynx 商业失败的最直接原因。**《California Games》（Epyx 1989，附赠）、《Klax》（Atari 1990）、《Blue Lightning》（Epyx 1989，战机射击代表作）、《Battlewheels》（1993，ComLynx 连线车战先驱）、《Lemmings》（Atari 1994）** ——技术示范不错但「杀手级」独占不足，1989-1995 年 Game Boy 已建立 Tetris + Pokémon 前的地基（《Super Mario Land》、《Metroid II》、《Final Fantasy Legend》），Lynx 没办法跟。

商业上 Lynx 全球累计约 **300 万台**——只有同期 Game Boy 1.18 亿的 **2.5%**。1995 年 Atari Corp 退出主机事业（被 JTS 合并），**Lynx 是 Atari 最后一台掌机，也是 Atari 整体衰退史的缩影**——曾经 1980 年初定义家用主机产业（Atari 2600 主导 1977-1983）的公司，在掌机时代用一台技术上领先却商业上无解的机器告别舞台。但**「彩色背光 + 双向手柄 + 连线多人」这三个 Lynx 独创概念在后续 25 年都被验证为对的方向**——这是技术历史上「对的构想 + 错的时间 + 错的公司」的经典案例。
