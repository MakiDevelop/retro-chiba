1989 年 9 月，Atari 在北美推出 Lynx，售價 $179.95 美元。**這是史上第一台彩色掌機**——與初代 Game Boy 同月發行、比 Sega Game Gear 早一年。原始設計來自 **Epyx**（《California Games》《Summer Games》系列開發商，當時的運動模擬遊戲一線廠），原型機代號 *Handy*，1989 年 Epyx 財務危機後 Atari 接手量產。

技術上 Lynx 領先同期 Game Boy 一個世代。**MOS 6502 衍生 CPU @ 4 MHz** + 自製協處理器 **Suzy / Mikey**（前者負責 blitter 圖形運算、後者整合系統匯流排與音源），**4,096 同屏色彩色背光 LCD（160×102）**、64 KB RAM、4 通道立體聲、**ComLynx 序列埠連線（最多 8 台同時連線）**。獨特的硬體設計還包括**雙向控制器**（透過螢幕翻轉，左撇子可以鏡像握持主控制）—— Game Boy / Game Gear / 後續所有掌機都沒做這件事。

**最大命門是耗電**——**6 顆 AA 撐 4-5 小時**（vs Game Boy 4 顆 AA 撐 30 小時），這是彩色背光在 1989 年硬體成本下的物理限制（Game Gear 一年後也撞上同樣問題）。Atari 1991 年推出 Lynx II（更小、立體聲耳機輸出、可關背光省電），但根本問題未解。**Atari 公司本身正在崩塌**——1989 年 Atari Corp 已被任天堂 NES（家用機 90% 市佔）和 Sega Genesis 雙重夾擊，研發資源不足、第三方信心崩潰。

軟體陣容稀薄是 Lynx 商業失敗的最直接原因。**《California Games》（Epyx 1989，內附）、《Klax》（Atari 1990）、《Blue Lightning》（Epyx 1989，戰機射擊代表作）、《Battlewheels》（1993，ComLynx 連線車戰先驅）、《Lemmings》（Atari 1994）** ——技術示範不錯但「殺手級」獨佔不足，1989-1995 年 Game Boy 已建立 Tetris + Pokémon 前的地基（《Super Mario Land》、《Metroid II》、《Final Fantasy Legend》），Lynx 沒辦法跟。

商業上 Lynx 全球累計約 **300 萬台**——只有同期 Game Boy 1.18 億的 **2.5%**。1995 年 Atari Corp 退出主機事業（被 JTS 合併），**Lynx 是 Atari 最後一台掌機，也是 Atari 整體衰退史的縮影**——曾經 1980 年初定義家用主機產業（Atari 2600 主導 1977-1983）的公司，在掌機時代用一台技術上領先卻商業上無解的機器告別舞台。但**「彩色背光 + 雙向控制器 + 連線多人」這三個 Lynx 獨創概念在後續 25 年都被驗證為對的方向**——這是技術歷史上「對的構想 + 錯的時間 + 錯的公司」的經典案例。
