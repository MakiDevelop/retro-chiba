1999 年 3 月 16 日，SNK 在日本推出 Neo Geo Pocket Color，售價 8,900 円——這是 1998 年原版黑白 Neo Geo Pocket 一年後的彩色升級版。**SNK 想做的是「把街機 Neo Geo MVS 的格鬥血統壓縮進掌機」**——畢竟 Neo Geo（1990 街機 + 家用機）讓 SNK 在 KOF / Samurai Shodown / Fatal Fury / Metal Slug 系列裡建立了「街機格鬥之王」的地位，這個 IP 庫往掌機搬是合理的延伸。

技術上 NGPC 在 1999 年規格凶猛。**東芝 TLCS-900H @ 6.144 MHz 16-bit 嵌入式 CPU**、**40,960 色 STN-LCD（160×152）**——是同期 Game Boy Color 56 色的 **7 倍**、T6W28 PSG 6 通道音源、ROM 卡帶（最大 16 MB）、**2 顆 AA 撐 40 小時**（比 Game Boy 本體還省電）。**最具靈魂的硬體是「8 向微動搖桿」**——SNK 把街機規格的 microswitch 搖桿（按下去會「咔嗒」響的格鬥用搖桿）直接塞到掌機上，**這是格鬥遊戲第一次在掌機上「真的可玩」**——之前 Game Boy / Game Gear 的十字鍵根本做不出格鬥輸入。

**軟體上 NGPC 的命脈就是格鬥遊戲**。**《SNK vs. Capcom: Match of the Millennium》（SNK 1999）是史上首見的格鬥遊戲跨公司對戰作**——SNK / Capcom 兩家當年是格鬥遊戲業界宿敵（KOF vs Street Fighter、Fatal Fury vs Darkstalkers），這款遊戲讓 Ryu / Ken / Terry / Mai 同台對戰，是格鬥遊戲史上的標誌性事件。其他強檔包括 **《King of Fighters R-2》（SNK 1999）、《Samurai Shodown! 2》（SNK 1999）、《Sonic the Hedgehog Pocket Adventure》（SNK/Sega 1999，跨家用機 Sonic 友好移植）、《Cool Boarders Pocket》（Sunsoft 2000）**。Sonic Pocket Adventure 是 Sega 唯一一次在「非 Sega 自家掌機」獨家發行 Sonic 遊戲——這是 1999 年 Sega 自家 Game Gear 已停產、Dreamcast 主推前的轉型期。

**SNK 1999-2001 年正在崩塌**。1999 年 SNK 母公司**新世界貿易**已財務困難，2001 年 SNK 正式破產（被 Aruze 收購），格鬥遊戲業界整體也在 1998-2002 年因 PS1 + 3D 格鬥（鐵拳 / 死或生 / 靈魂能力）轉型期下滑。NGPC 的時間點極差——上市同年 9 月**Pokémon 黃**在 Game Boy / Game Boy Color 上發行（Pokémon 紅綠藍黃 1996-1999 賣 4,500 萬份）+ Game Boy Color（1998）整合 Pokémon + Tetris + Mario Land 三大強檔，**NGPC 等於對戰整個任天堂第一方陣容 + Pokémon 怪物現象**。

商業上 NGPC 全球累計 **200 萬台**（含 1998 黑白原版）——只有同期 Game Boy Color（5,000 萬台）的 **4%**。**2000 年 SNK 退出掌機業務**——比 SNK 整體破產（2001）早一年。**這是 SNK 唯一一台掌機，也是 SNK 硬體事業的句點**——之後 SNK 復活後（2003 年改稱 SNK Playmore，2016 年回到 SNK）只做軟體與街機板，再無 dedicated handheld。

**但 NGPC 的文化地位遠超銷量**——格鬥遊戲社群至今奉為「**史上最棒的掌機格鬥遊戲機**」（Retro Achievements、X 復古社群、X 上 fighting game discourse 公認），原因是**沒有任何後續掌機（GBA / DS / PSP / 3DS / Switch）再做過 8 向微動搖桿**。NGPC 是格鬥遊戲掌機這個 niche 的唯一峰值，**短壽但 cult 程度極高 + 收藏品價值穩定**——完整盒裝《SNK vs. Capcom: Match of the Millennium》在 2025 年 eBay 約 $300-500 美元。
