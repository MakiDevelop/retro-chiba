1999 年 3 月 16 日，SNK 在日本推出 Neo Geo Pocket Color，售价 8,900 日元——这是 1998 年原版黑白 Neo Geo Pocket 一年后的彩色升级版。**SNK 想做的是「把街机 Neo Geo MVS 的格斗血统压缩进掌机」**——毕竟 Neo Geo（1990 街机 + 家用机）让 SNK 在 KOF / 侍魂 / 饿狼传说 / 合金弹头系列里建立了「街机格斗之王」的地位，这个 IP 库往掌机搬是合理的延伸。

技术上 NGPC 在 1999 年规格凶猛。**东芝 TLCS-900H @ 6.144 MHz 16-bit 嵌入式 CPU**、**40,960 色 STN-LCD（160×152）**——是同期 Game Boy Color 56 色的 **7 倍**、T6W28 PSG 6 通道音源、ROM 卡带（最大 16 MB）、**2 颗 AA 撑 40 小时**（比 Game Boy 本体还省电）。**最具灵魂的硬件是「8 向微动摇杆」**——SNK 把街机规格的 microswitch 摇杆（按下去会「咔嗒」响的格斗用摇杆）直接塞到掌机上，**这是格斗游戏第一次在掌机上「真的可玩」**——之前 Game Boy / Game Gear 的十字键根本做不出格斗输入。

**软件上 NGPC 的命脉就是格斗游戏**。**《SNK vs. Capcom: Match of the Millennium》（SNK 1999）是史上首见的格斗游戏跨公司对战作**——SNK / Capcom 两家当年是格斗游戏业界宿敌（KOF vs 街霸、饿狼传说 vs Darkstalkers），这款游戏让 Ryu / Ken / Terry / Mai 同台对战，是格斗游戏史上的标志性事件。其他强档包括 **《King of Fighters R-2》（SNK 1999）、《侍魂! 2》（SNK 1999）、《刺猬索尼克 Pocket Adventure》（SNK/Sega 1999，跨家用机 Sonic 友好移植）、《Cool Boarders Pocket》（Sunsoft 2000）**。Sonic Pocket Adventure 是 Sega 唯一一次在「非 Sega 自家掌机」独家发行 Sonic 游戏——这是 1999 年 Sega 自家 Game Gear 已停产、Dreamcast 主推前的转型期。

**SNK 1999-2001 年正在崩塌**。1999 年 SNK 母公司**新世界贸易**已财务困难，2001 年 SNK 正式破产（被 Aruze 收购），格斗游戏业界整体也在 1998-2002 年因 PS1 + 3D 格斗（铁拳 / 生死格斗 / 灵魂能力）转型期下滑。NGPC 的时间点极差——上市同年 9 月**Pokémon 黄**在 Game Boy / Game Boy Color 上发行（Pokémon 红绿蓝黄 1996-1999 卖 4,500 万份）+ Game Boy Color（1998）整合 Pokémon + Tetris + Mario Land 三大强档，**NGPC 等于对战整个任天堂第一方阵容 + Pokémon 怪物现象**。

商业上 NGPC 全球累计 **200 万台**（含 1998 黑白原版）——只有同期 Game Boy Color（5,000 万台）的 **4%**。**2000 年 SNK 退出掌机业务**——比 SNK 整体破产（2001）早一年。**这是 SNK 唯一一台掌机，也是 SNK 硬件事业的句点**——之后 SNK 复活后（2003 年改称 SNK Playmore，2016 年回到 SNK）只做软件与街机板，再无 dedicated handheld。

**但 NGPC 的文化地位远超销量**——格斗游戏社群至今奉为「**史上最棒的掌机格斗游戏机**」（Retro Achievements、X 复古社群、X 上 fighting game discourse 公认），原因是**没有任何后续掌机（GBA / DS / PSP / 3DS / Switch）再做过 8 向微动摇杆**。NGPC 是格斗游戏掌机这个 niche 的唯一峰值，**短寿但 cult 程度极高 + 收藏品价值稳定**——完整盒装《SNK vs. Capcom: Match of the Millennium》在 2025 年 eBay 约 $300-500 美元。
