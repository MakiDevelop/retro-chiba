2001 年 3 月 21 日，任天堂在日本推出 Game Boy Advance（GBA），售价 9,800 日元。**这是横井军平 1997 年离世后，任天堂掌机部门的第一个世代**——由横井派传承的 R&D1 部门主导，冈田智继任。GBA 是任天堂掌机架构的历史性转折——**从 1989 年起的 8-bit Sharp CPU 系谱全面换成 32-bit ARM**，但仍保留横井「水平思考」的核心精神（成熟技术、长电池、轻薄外型）。

技术规格上 GBA 是「**SFC 级 2D 图形掌机化**」的具体实现。**ARM7TDMI @ 16.78 MHz**（32-bit 比起 GBC 的 8 MHz 8-bit 快了一个世代）、**32,768 色 STN-LCD**、240×160 横向屏幕（GB 系列首次采用横向比例）、双通道 PCM 音源。**最关键的工程决策：芯片内仍保留一颗 Sharp LR35902（GBC CPU）作为向下兼容用**。这意味 GBA **能跑从 1989 年初代 Game Boy、1998 年 GBC 到 2001 年 GBA 自身的全部三代卡带**——**12 年时间跨度的卡带兼容**，是当时掌机产业最完整的向下兼容支援。

软件阵容是 GBA 真正的黄金。**《口袋妖怪 红宝石/蓝宝石》**（2002）系列销量 1,640 万份、**《口袋妖怪 绿宝石》**（2004）610 万份、**《恶魔城：晓月圆舞曲》**（2003，IGA 主导，恶魔城 Metroidvania 范本之一）、**《银河战士 Fusion》**（2002）、**《Mother 3》**（2006，糸井重里集大成作）、**《最终幻想战略版 Advance》**（2003，松野泰己回 Square 后第一作）、**《高级战争》**（2001，Intelligent Systems，被《火焰之纹章》系列承袭的 turn-based 战术设计）。**整个世代的第三方阵容堪称掌机历史最完整的时代**。

外型上 GBA 出了三个主要 SKU。**初代 GBA**（2001 横向，无背光）→ **GBA SP**（2003 翻盖小型 + **内置锂充电池** + **后期版内建前光照明**）→ **GBA Micro**（2005，超薄迷你版，主打高端配件市场）。**SP 是 GBA 世代最畅销的型号**——翻盖设计影响后来 NDS、3DS 一路到 2017 年。**Micro 是任天堂掌机史上最小的机型**，但因为已是 GBA 末期、又面对 2004 年才刚上市的 NDS，**Micro 销量极低、现在 retro 市场价格反而最高**。

中国大陆 GBA + 烧录卡（如「**金手指 GBA Cart**」、「**EZ-Flash 2**」）是 80/90 后玩家最普及的「**掌机 + 海量游戏**」入门组合——一片人民币 100-200 元能存 32-128 款游戏。**「Pokémon 红宝石 + EZ-Flash + 中文化补丁」是中国掌机黄金时代的核心记忆**。台湾水货市场 GBA 在中华商场后期、光华商场是主流商品。

商业上 GBA 全球累计 **8,151 万台**（含 GBA + SP + Micro）——比 GB+GBC 的 1.187 亿略低，但是任天堂掌机家族中 NDS 之前最高销量。**生产期 2001-2010 年，9 年寿命**——比 GBC 还长。**GBA 是任天堂从横井哲学过渡到双屏 NDS 哲学的桥梁机型**——既保留 8-bit 时代的全部资产，又把硬件架构升级到 32-bit ARM 时代，为 2004 年的 NDS 铺好兼容性与第三方阵容的双重基础。
