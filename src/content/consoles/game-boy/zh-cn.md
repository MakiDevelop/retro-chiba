1989 年 4 月 21 日，任天堂在日本推出 Game Boy，售价 12,500 日元。**设计者是横井军平**——任天堂史上最有影响力的工程师之一，1980 年代主导 Game & Watch 系列、Famicom 手柄十字键发明者。横井提出的设计哲学是**「枯れた技術の水平思考」（成熟技术的水平思考）**——不追新规格，反而用业界已验证可量产、便宜、可靠的旧技术组合出新体验。Game Boy 是这个哲学最完整的实践——**省规格、长电池、强内容**——这套思维在 30 年后的 Switch 上仍然能看见。

技术上 Game Boy 在 1989 年是**刻意保守**的选择。**4 灰阶 STN-LCD 而非彩色**（同期 Atari Lynx 1989 已有彩色）、**无背光**（户外才能玩、室内须开灯）、**Sharp LR35902 CPU @ 4.19 MHz**（Z80 派生混 8080 指令的廉价变种）。但这些「保守」决策直接带来决定性的优势——**4 颗 AA 电池能跑 30 小时**。同期 Atari Lynx 才 4-5 小时、Game Gear 也只 3-4 小时。**Game Boy 出门整个礼拜不用充电，这个体验差距是规格战的反向胜利**。

1989 年 6 月，**Tetris 随机附赠**——任天堂从苏联设计师 Alexey Pajitnov 手上争取到的 Game Boy 独家版权（这场跨国授权战本身就是冷战末期商业史的一段传奇）。**Tetris 是 Game Boy 史上最关键的决策**：不是任何任天堂的招牌 IP（Mario、Zelda 都晚于它），而是一款**苏联出品的方块俄罗斯游戏**让全球家长、上班族、学生都拿起 Game Boy。**「掌机 = 通勤族 + 家庭主妇的口袋玩具」这个品类就是 Tetris + Game Boy 共同定义的**。

但真正让 Game Boy 从「6-7 年寿命的成功掌机」变成**史上现象级平台**的是 1996 年的**《Pokémon 红/绿》**（GameFreak / 田尻智）。当时 Game Boy 已经 7 岁、业界普遍认为快被 Game Boy Color 取代——**结果 Pokémon 在日本爆红、1998 年北美登陆后变成全球文化现象**。**Pokémon 红/绿/蓝/黄在 Game Boy 上累计卖 4,600 万份**，把 Game Boy 平台寿命延长到 1999 年 GBC 上市才结束。**Pokémon 是任天堂史上最大的「补完」——拯救了一台已经老去的硬件**。

中国大陆 Game Boy 因 4 颗 AA 电池与卡带昂贵，没有像 FC「小霸王」山寨机那样普及——多数 80/90 后玩家的 Game Boy 第一接触是 Pokémon 红/绿的盗版卡带（1996-1998 年间透过港台流入），口袋妖怪是大陆中小学生这一代的童年共同记忆。台湾水货 Game Boy 是**中华商场二楼最普及的掌机**——**水货日版 + 卡带可改机跑欧美 ROM** 是 1990 年代台北少年的标配。

商业数字惊人。**Game Boy + Game Boy Pocket 累计 1 亿 1,869 万台**（Nintendo 2003 累计，含 1996 年小型化 Pocket 版）——是**史上第二高销掌机，仅次任天堂 DS（1.54 亿）**。产品寿命**从 1989 到 2003 年正式停产，14 年**——比同期任何家用机都久。**Game Boy 证明了横井「水平思考」哲学的长期正确性**——**用便宜、可靠、长电池的硬件承载 Pokémon 这种大爆款内容，比追新规格更值钱**。这个道理在 2017 年的 Switch 上再次被证实。横井 1997 年离开任天堂创立 Koto Laboratory（后来做 WonderSwan，但公司不久即倒闭）、**1997 年 10 月于东名高速公路车祸身亡，享年 56 岁**。横井没看见 Pokémon 把他的 Game Boy 复活、也没看见 Switch 把他的哲学再现。
