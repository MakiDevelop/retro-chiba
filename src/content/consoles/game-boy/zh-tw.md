1989 年 4 月 21 日，任天堂在日本推出 Game Boy，售價 12,500 円。**設計者是橫井軍平**——任天堂史上最有影響力的工程師之一，1980 年代主導 Game & Watch 系列、Famicom 控制器十字鍵發明者。橫井提出的設計哲學是**「枯れた技術の水平思考」（成熟技術的水平思考）**——不追新規格，反而用業界已驗證可量產、便宜、可靠的舊技術組合出新體驗。Game Boy 是這個哲學最完整的實踐——**省規格、長電池、強內容**——這套思維在 30 年後的 Switch 上仍然能看見。

技術上 Game Boy 在 1989 年是**刻意保守**的選擇。**4 灰階 STN-LCD 而非彩色**（同期 Atari Lynx 1989 已有彩色）、**無背光**（戶外才能玩、室內須開燈）、**Sharp LR35902 CPU @ 4.19 MHz**（Z80 衍生混 8080 指令的廉價變種）。但這些「保守」決策直接帶來決定性的優勢——**4 顆 AA 電池能跑 30 小時**。同期 Atari Lynx 才 4-5 小時、Game Gear 也只 3-4 小時。**Game Boy 出門整個禮拜不用充電，這個體驗差距是規格戰的反向勝利**。

1989 年 6 月，**Tetris 隨機附贈**——任天堂從蘇聯設計師 Alexey Pajitnov 手上爭取到的 Game Boy 獨家版權（這場跨國授權戰本身就是冷戰末期商業史的一段傳奇）。**Tetris 是 Game Boy 史上最關鍵的決策**：不是任何任天堂的招牌 IP（Mario、Zelda 都晚於它），而是一款**蘇聯出品的方塊俄羅斯遊戲**讓全球家長、上班族、學生都拿起 Game Boy。**「掌機 = 通勤族 + 家庭主婦的口袋玩具」這個品類就是 Tetris + Game Boy 共同定義的**。

但真正讓 Game Boy 從「6-7 年壽命的成功掌機」變成**史上現象級平台**的是 1996 年的**《Pokémon 紅/綠》**（GameFreak / 田尻智）。當時 Game Boy 已經 7 歲、業界普遍認為快被 Game Boy Color 取代——**結果 Pokémon 在日本爆紅、1998 年北美登陸後變成全球文化現象**。**Pokémon 紅/綠/藍/黃在 Game Boy 上累計賣 4,600 萬份**，把 Game Boy 平台壽命延長到 1999 年 GBC 上市才結束。**Pokémon 是任天堂史上最大的「補完」——拯救了一台已經老去的硬體**。

對台灣的孩子，Game Boy 是**中華商場二樓最普及的掌機**——**水貨日版 + 卡帶可改機跑歐美 ROM** 是 1990 年代台北少年的標配。**中華商場 1992 年拆除前最後幾年的記憶就是擠滿學生的 Game Boy 卡帶櫃**。中國大陸 Game Boy 因 4 顆 AA 電池與卡帶昂貴，沒有像 FC 山寨機那樣普及——多數中國 80/90 後玩家的 Game Boy 第一接觸是 Pokémon 紅/綠的盜版卡帶（1996-1998 年間透過港台流入）。

商業數字驚人。**Game Boy + Game Boy Pocket 累計 1 億 1,869 萬台**（Nintendo 2003 累計，含 1996 年小型化 Pocket 版）——是**史上第二高銷掌機，僅次任天堂 DS（1.54 億）**。產品壽命**從 1989 到 2003 年正式停產，14 年**——比同期任何家用機都久。**Game Boy 證明了橫井「水平思考」哲學的長期正確性**——**用便宜、可靠、長電池的硬體承載 Pokémon 這種大爆款內容，比追新規格更值錢**。這個道理在 2017 年的 Switch 上再次被證實。橫井 1997 年離開任天堂創立 Koto Laboratory（後來做 WonderSwan，但公司不久即倒閉）、**1997 年 10 月於東名高速公路車禍身亡，享年 56 歲**。橫井沒看見 Pokémon 把他的 Game Boy 復活、也沒看見 Switch 把他的哲學再現。
