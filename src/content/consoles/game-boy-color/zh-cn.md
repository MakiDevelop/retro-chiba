1998 年 10 月 21 日，任天堂在日本推出 Game Boy Color（GBC），售价 8,900 日元。**这是任天堂史上最 disciplined 的中期升级案**——CPU 双倍速（4.19 MHz → 8 MHz）、56 色 STN-LCD（取代 4 灰阶）、32 KB RAM（4 倍）、外型小型化。**但故意保留** Sharp 同系 CPU 架构与 Game Pak 卡带物理规格——这个「保留」的决定是 GBC 整代成功的关键。

最关键的设计决策是**完全向下兼容初代 Game Boy 卡带**，而且不只是「能跑」——**1989-98 年累积的所有卡带在 GBC 上自动变彩色**。任天堂为每款 1989 年起的初代 Game Boy 黑白卡带**内建 12 种预设彩色配方**（Mario、Pokémon、Tetris 等热门作各有独家配色）。**这意味着旧用户买 GBC 升级的瞬间，他们十年累积的卡带库变彩色** ——零学习成本、零卡带迁移、零内容真空期。**这是史上最成功的硬件向下兼容战略，也是 Switch 2 完整继承 Switch 1 的祖先**。

杀手级应用是**《口袋妖怪 金/银》**（1999）——田尻智的续作把现实世界 24 小时时钟整合进游戏（早晨/黄昏/夜晚不同宝可梦出现、月亮石只在月圆出现），**这个内建即时时钟 RTC 是当时掌机从未做过的硬件 + 软件整合**。**金/银全球销售 2,310 万份**，是 GBC 整代的销量火车头。其他重要作品包括 Capcom Flagship 开发的**塞尔达神谕系列**（2001）、Wario Land 3、勇者斗恶龙 III GBC 移植版。

**中国大陆 GBC + Pokémon 金/银盗版卡是 80 后童年的共同记忆**——多数人第一次接触宝可梦就是 GBC 紫色机 + 不会中文化的英文版盗版卡带。台湾中华商场后期、光华商场 GBC 跟金/银盗版卡是少年标配。

GBC 累计销量与初代 Game Boy 合并计算（任天堂从未拆分），合计 **1 亿 1,869 万台**。GBC 单独估计约 **5,000 万台**——成为**任天堂 21 世纪前最后一代 16-color 级别掌机**。2001 年 Game Boy Advance（32-bit ARM 架构）接棒，**完全打破 GBC 与初代 Game Boy 共用的 8-bit Sharp CPU 体系**——这是任天堂掌机史上最大的单次架构断层，也意味 GBC + 初代 Game Boy 共享的卡带 ROM 库从此进入 retro 收藏品阶段。
