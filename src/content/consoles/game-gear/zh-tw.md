1990 年 10 月 6 日，Sega 在日本推出 Game Gear，售價 19,800 円。**這是 Sega 第一次正面挑戰任天堂的掌機霸權**——也是 Sega 在主場（家用機）打不贏 Nintendo 後，把戰場搬到掌機的嘗試。設計策略很直接：**把 Master System 的硬體（Z80 + VDP + PSG）整個塞進掌機**——也是後續 Sony PSP 用 PS2 級規格、PS Vita 用 PS3 級規格的同一條「家用機小型化」邏輯。

技術上 Game Gear 在「畫面」這個維度上勝 Game Boy。**Zilog Z80 @ 3.58 MHz**（與 Master System 同核心）+ **彩色背光 STN-LCD 160×144、同屏 32 色**（vs Game Boy 的 4 灰階黑綠 LCD），TI SN76489 PSG 4 通道音源、ROM 卡帶 + **Master System 卡帶轉接器**（可玩家用機完整 SMS 軟體庫）。**「彩色 + 背光」這個畫面優勢在 1990 年的賣場展示效果壓倒 Game Boy**——Sega 廣告語「Welcome to the Next Level」直接嘲諷 Nintendo 的「黑白蛋黃綠」。

**但耗電是 Game Gear 的致命傷**——**6 顆 AA 撐 3-5 小時**，是同期 Game Boy 4 顆 AA 撐 30 小時的 **1/6**。1990 年那個時代「**包包塞 12 顆 AA**」是 Game Gear 玩家的標配景觀，車旅 / 通勤實戰中根本撐不過半天。Sega 1991 年推出 TV Tuner 配件（外接收音器讓 Game Gear 變成攜帶式電視）+ 廣告主打「Bring Your TV With You」——但 TV Tuner 又進一步把耗電壓到 1.5 小時，**品牌定位反而越推越混亂**（到底是掌機？還是攜帶電視？）。

軟體上 Game Gear 有 **《Sonic the Hedgehog》（Sega 1991）、《Columns》（Sega 1990，俄羅斯方塊式消除）、《Shining Force Gaiden》（Sega 1993，Shining Force 系列移植）、《Sonic Drift》（Sega 1994）、《OutRun》（Sega 1991 移植）** —— 第一方陣容不弱，但**第三方支援遠不及 Game Boy**（Sega 在 Master System 時代第三方信心已被 Nintendo SES 鎖卡帶政策打散）。Sonic 系列雖然強檔，但 Game Boy 同期有 Pokémon 紅綠（1996）+ Tetris + Mario Land 三大強檔——軟體陣容差距才是 Game Gear 銷量壓不過的根本原因。

商業上 Game Gear **全球累計 1,062 萬台**（Sega 1997 年停產時數據）——只有同期 Game Boy 1.18 億的 **9%**。**這是 Sega 在 Nintendo 主場（掌機）的徹底失敗**——Sega 後續再也沒有推 dedicated handheld（Nomad 是 Genesis 攜帶版、不算原生掌機），1999 年起把資源全押 Dreamcast。**Game Gear 之後的歷史可以視為「Sega 從硬體業務逐步退場」的前奏**——2001 年 Dreamcast 停產、Sega 退出主機業務後，Game Gear 也成了 Sega 硬體史上的一個句點。
