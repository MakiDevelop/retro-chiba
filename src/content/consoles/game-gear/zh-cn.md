1990 年 10 月 6 日，Sega 在日本推出 Game Gear，售价 19,800 日元。**这是 Sega 第一次正面挑战任天堂的掌机霸权**——也是 Sega 在主场（家用机）打不赢 Nintendo 后，把战场搬到掌机的尝试。设计策略很直接：**把 Master System 的硬件（Z80 + VDP + PSG）整个塞进掌机**——也是后续 Sony PSP 用 PS2 级规格、PS Vita 用 PS3 级规格的同一条「家用机小型化」逻辑。

技术上 Game Gear 在「画面」这个维度上胜 Game Boy。**Zilog Z80 @ 3.58 MHz**（与 Master System 同核心）+ **彩色背光 STN-LCD 160×144、同屏 32 色**（vs Game Boy 的 4 灰阶黑绿 LCD），TI SN76489 PSG 4 通道音源、ROM 卡带 + **Master System 卡带转接器**（可玩家用机完整 SMS 软件库）。**「彩色 + 背光」这个画面优势在 1990 年的卖场展示效果压倒 Game Boy**——Sega 广告语「Welcome to the Next Level」直接嘲讽 Nintendo 的「黑白蛋黄绿」。

**但耗电是 Game Gear 的致命伤**——**6 颗 AA 撑 3-5 小时**，是同期 Game Boy 4 颗 AA 撑 30 小时的 **1/6**。1990 年那个时代「**包里塞 12 颗 AA**」是 Game Gear 玩家的标配景观，车旅 / 通勤实战中根本撑不过半天。Sega 1991 年推出 TV Tuner 配件（外接收音器让 Game Gear 变成便携式电视）+ 广告主打「Bring Your TV With You」——但 TV Tuner 又进一步把耗电压到 1.5 小时，**品牌定位反而越推越混乱**（到底是掌机？还是便携电视？）。

软件上 Game Gear 有 **《刺猬索尼克》（Sega 1991）、《Columns》（Sega 1990，俄罗斯方块式消除）、《Shining Force Gaiden》（Sega 1993，光明力量系列移植）、《Sonic Drift》（Sega 1994）、《OutRun》（Sega 1991 移植）** —— 第一方阵容不弱，但**第三方支援远不及 Game Boy**（Sega 在 Master System 时代第三方信心已被 Nintendo SES 锁卡带政策打散）。Sonic 系列虽然强档，但 Game Boy 同期有 Pokémon 红绿（1996）+ Tetris + Mario Land 三大强档——软件阵容差距才是 Game Gear 销量压不过的根本原因。

商业上 Game Gear **全球累计 1,062 万台**（Sega 1997 年停产时数据）——只有同期 Game Boy 1.18 亿的 **9%**。**这是 Sega 在 Nintendo 主场（掌机）的彻底失败**——Sega 后续再也没有推 dedicated handheld（Nomad 是 Genesis 便携版、不算原生掌机），1999 年起把资源全押 Dreamcast。**Game Gear 之后的历史可以视为「Sega 从硬件业务逐步退场」的前奏**——2001 年 Dreamcast 停产、Sega 退出主机业务后，Game Gear 也成了 Sega 硬件史上的一个句点。
